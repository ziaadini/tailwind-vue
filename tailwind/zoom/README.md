### props

| name           | Type    | Default | Description                                                |
| -------------- | ------- | ------- | ---------------------------------------------------------- |
| src            | string  | ""      | image source to show                                   |
| scale          | number  | 1.5     | scale number to scale the image by that number (1.5,2,3,4) |
| maskSrc        | string  | ""      | image mask source to show                                   |
